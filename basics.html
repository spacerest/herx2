general notes

    html–css differences

  • hypertext markup language gives content structure & meaning
  • cascading style sheets is a presentation language that styles the appearance of content

    html things

  • elements are designators of structure and content (h1 - h6 etc.)
  • tags
  • attributes are properties that give extra info about an element (href="__")

    self–closing elements

  • < br >
  • < img >
  • < meta >
  • < wbr >
  • < embed >
  • < input >
  • < param >
  • < hr >
  • < link >
  • < source >

    common css terms

  • selectors designate exactly which elements in corresponding html to target (i.e. color, size, position)
    • class selectors "."
    • id selectors "#"
  • properties determine which style is applied to an element
  • values determine the behavior (style?) of an element

    css resets

  • eric meyer's reset — pretty basic reset which includes new html5 elements
  • normalize.css — more advanced reset created by nicolas gallagher and jonathan neal
